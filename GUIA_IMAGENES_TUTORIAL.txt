═══════════════════════════════════════════════════════════════
  GUÍA: COMO AÑADIR IMÁGENES AL TUTORIAL
═══════════════════════════════════════════════════════════════

El archivo "tutorial_BBCode_con_imagenes.txt" tiene placeholders
de imágenes que debes reemplazar con URLs reales.

═══════════════════════════════════════════════════════════════
  PASO 1: SUBIR IMÁGENES A TU FORO
═══════════════════════════════════════════════════════════════

OPCIÓN A - Usar el alojamiento de ForoActivo:

1. Ve a tu foro → "Galería" o "Alojamiento de imágenes"
2. Crea un álbum nuevo llamado "Tutorial uBlock"
3. Sube las capturas de pantalla que harás
4. Copia la URL de cada imagen
5. Reemplaza los placeholders en el tutorial

OPCIÓN B - Servicios externos (más fácil):

• Imgur (imgur.com) - ARRASTRA y suelta las imágenes
• Postimages (postimages.org)
• Servimg (servimg.com)

═══════════════════════════════════════════════════════════════
  PASO 2: IMÁGENES NECESARIAS
═══════════════════════════════════════════════════════════════

Estas son las capturas que necesitas hacer:

1. UBLOG_LOGO.png
   → Logo de uBlock Origin (buscar en Google)

2. CHROME_PROPIEDADES.png
   → Click derecho en icono Chrome → Propiedades

3. DESTINO_ANTES.png
   → Campo "Destino" con el valor original

4. DESTINO_DESPUES.png
   → Campo "Destino" MODIFICADO con el código

5. WEBSTORE_BUSCAR.png
   → Chrome Web Store buscando "uBlock Origin"

6. UBLOCK_INSTALAR.png
   → Página de uBlock con botón "Añadir a Chrome"

7. UBLOCK_ICONO_BAR.png
   → Barra de Chrome mostrando icono 🛡️

8. UBLOCK_PANEL.png
   → Panel de uBlock abierto

9. UBLOCK_FILTROS.png
   → Pestaña "Lista de filtros" marcada

10. PAGINA_CON_ANUNCIOS.png
    → Ejemplo de página LLENA de anuncios

11. PAGINA_SIN_ANUNCIOS.png
    → Misma página CON uBlock activado

═══════════════════════════════════════════════════════════════
  PASO 3: REEMPLAZAR URLs EN EL TUTORIAL
═══════════════════════════════════════════════════════════════

En el archivo "tutorial_BBCode_con_imagenes.txt" verás códigos así:

    [img]https://i.imgur.com/UBLOCK_LOGO.png[/img]

Reemplázalos por las URLs reales:

    [img]https://i.imgur.com/TuURLReal.jpg[/img]

═══════════════════════════════════════════════════════════════
  BBCode PARA IMÁGENES
═══════════════════════════════════════════════════════════════

IMAGEN SIMPLE:
    [img]URL_DE_LA_IMAGEN[/img]

IMAGEN CON TAMAÑO (si es muy grande):
    [img width=500]URL_DE_LA_IMAGEN[/img]

IMAGEN CENTRADA:
    [center][img]URL_DE_LA_IMAGEN[/img][/center]

IMAGEN CON ENLACE (clic para ampliar):
    [url=URL_AMPLIADA][img]URL_PEQUEÑA[/img][/url]

═══════════════════════════════════════════════════════════════
  CONSEJOS PARA LAS CAPTURAS
═══════════════════════════════════════════════════════════════

• Usa la herramienta "Recortes" de Windows (Win+Shift+S)
• Recorta solo la parte importante
• No captures información personal
• Guarda en formato PNG para mejor calidad
• Tamaño recomendado: 800-1000px de ancho máximo

═══════════════════════════════════════════════════════════════
  EJEMPLO RÁPIDO
═══════════════════════════════════════════════════════════════

1. Presiona Win+Shift+S
2. Selecciona el área a capturar
3. Abre Paint y pega (Ctrl+V)
4. Guarda como PNG
5. Sube a Imgur
6. Copia el link directo
7. Reemplaza en el tutorial
